# 第11章 未来信息综合技术

- **11.1 信息物理系统 (CPS)**
  - **11.1.1 概念**
    - **本质**：计算、通信、控制 (3C) 与物理过程的深度融合
    - **定义**：物理空间与信息空间中人、机、物、环境、信息相互映射、交互、协同的复杂系统
  - **11.1.2 体系架构**
    - **单元级**：最小不可分割单元 (闭环：感知-分析-决策-控制)
    - **系统级**：多个单元级互联 (互操作、协同控制)
    - **SoS级 (系统之系统)**：跨系统/平台互联 (数据汇聚、运营优化)
  - **11.1.3 技术体系**
    - **一硬**：感知和自动控制 (传感器、虚实融合控制)
    - **一软**：工业软件 (核心规则固化)
    - **一网**：工业网络 (互联互通载体)
    - **一平台**：工业云和智能服务平台 (数据汇聚/服务)
  - **11.1.4 应用场景**
    - **智能设计** (仿真/数字样机)、**智能生产** (设备/生产管理/柔性制造)、**智能服务** (健康管理/预维护)、**智能应用** (无人装备/产业链互动)

- **11.2 人工智能 (AI)**
  - **11.2.1 概念**
    - **弱人工智能** (专用智能, 无意识)、**强人工智能** (通用, 类人/非类人思维)
  - **11.2.2 关键技术**
    - **NLP (自然语言处理)**：机器翻译、语义理解、问答系统
    - **CV (计算机视觉)**：图像提取/分析/理解
    - **KG (知识图谱)**：结构化语义知识库 (节点+边)
    - **HCI (人机交互)**：语音/情感/体感/脑机交互
    - **VR/AR**：虚拟/增强现实
    - **机器学习 (ML)**：
      - **模式**：监督 (有标)、无监督 (无标)、半监督、强化 (反馈)
      - **方法**：传统 (SVM/决策树/贝叶斯)、深度学习 (CNN/RNN, 自动特征提取)
      - **其他**：迁移学习、主动学习、演化学习

- **11.3 机器人技术**
  - **11.3.1 发展阶段**
    - **第一代**：示教再现型 (重复动作)
    - **第二代**：感觉型 (有感知无智能)
    - **第三代**：智能型 (自主决策)
  - **11.3.2 机器人4.0核心技术**
    - **云-边-端协同计算**：算力分布、任务迁移
    - **持续学习与协同学习**：小样本建立能力 + 大数据共享进化
    - **知识图谱**：动态/个性化知识 + 感知决策结合
    - **场景自适应**：三维语义理解 + 行为预测
    - **数据安全**：隐私保护/传输存储安全
  - **11.3.3 分类**
    - **按控制**：操作、程序、示教再现、智能、综合
    - **按行业**：工业、服务、特殊领域

- **11.4 边缘计算**
  - **11.4.1 概念**
    - 靠近物/数据源头的网络边缘侧提供计算/存储/服务
    - **形态**：云边缘、边缘云、云化网关
  - **11.4.2 特点 (CROSS)**
    - **C**onnectivity (联接性), **R**eal-time (实时性), **O**ptimization (数据优化), **S**mart (智能), **S**ecurity (安全)
  - **11.4.3 边云协同 (6种)**
    - 资源、数据、智能、应用管理、业务管理、服务协同
  - **11.4.4 应用**
    - 智慧园区、安卓云/云游戏、视频监控、工业物联网、Cloud VR

- **11.5 数字孪生体 (Digital Twin)**
  - **11.5.1 定义**
    - 物理实体对象的数字模型，通过实测/仿真/分析实现实时感知/诊断/预测/调控
    - **类型**：原型体、实例体、聚合体
  - **11.5.2 关键技术**
    - **核心**：建模 (源头)、仿真 (验证/确认)、数字线程
    - **外围**：IoT, 云/大数据/AI
  - **11.5.3 应用**
    - **制造** (研发设计/生产制造)、**产业** (全产业链)、**城市** (智慧城市基础)、**战场** (辅助决策)

- **11.6 云计算与大数据**
  - **11.6.1 云计算**
    - **服务模式**：SaaS (软件即服务), PaaS (平台即服务), IaaS (基础设施即服务)
    - **部署模式**：公有云、私有云、社区云、混合云
  - **11.6.2 大数据**
    - **特征 (5V)**：Volume (大量), Velocity (高速), Variety (多样), Value (价值), Veracity (真实/复杂)
    - **处理流程**：获取/记录 -> 抽取/清洗 -> 集成/表示 -> 分析/建模 -> 解释/可视化 (数据出处/Provenance)
